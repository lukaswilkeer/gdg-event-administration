About

This GDG EA app started with a little script to enable developers the online inscription for events at a local GDG in Brazil.
As more features where added the app got more complete and impossible to maintain and develope by just one person. The need was felt to make it a OS project to look for other developers to help making a the app.
The project is hosted on Google Code as an open source project and is publicly accesible. The working app itselve is available (for the moment) only to member of the gtug.org domain.
